- About Me -

Hi, My name is Fábio Caldeira and I'm not a Professional Programmer, because I don't have any professional experience in the software field. I don't have a Degree in Software Engineering or any Degree at all.
But I'm a Self-Learner and with that I pretend to build my career as a Programmer.
I start learning Programming in 2016, almost a year ago, with Codecademy and SoloLearn. With the free courses I accomplished the first baby steps in the coding world, I assimilated HTML and CSS and started JS. Now I'm teaching myself, with help of tutorials and courses, C++, Java, Unity Game Engine and Blender.
I'm a focused guy, that's one of the best particularities about me. I never quit, I never give up because I believe in Myself. I'm following a plan to be able to overwrite the fact that I can't take a Software Engineering course. That plan is based in four stages, each of them with specific goals.
The first one is to learn web languages(html,css,js,mysql,...) and software languages(c++,c#,JAVA), I'm working on it.
The second, third and forth are a bunch of projects made to practice and Master the languages.
At this moment I'm seeking volunteer projects, indie projects or similar to start learning more about the community, the career I want to seek and to gain experience.
